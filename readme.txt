== Installation ==

Installing from WordPress
Follow the steps below to install inside Wordpress
*  Navigate to Appearance -> Plugins
*  Click Install Plugins and then Upload
*  Navigate to iTehemland-Logo-Showcase-for-wp.zip on your computer and click Install Now
*  To activate the newly installed plugin navigate to Appearance -> Plugins select the iThemeland Testimonial for WP  and press Activate
*  After install, activate the plugin.


Installing via FTP Server
Follow the steps below to install via FTP
*  Unzip iTehemland-Logo-Showcase-for-wp.zip to a desired location on your HDD. These extracted files will be uploaded later via FTP
*  Connect to your server via FTP
*  Navigate to /wp-content/plugins/ on server
*  Navigate to location of unzipped iTehemland-Logo-Showcase-for-wp.zip
*  Upload local folder /iTehemland-Logo-Showcase-foe-wp to remote /wp-content/plugins/
*  To activate the newly installed plugin navigate to Appearance -> Plugins select the iTehemland Logo Showcase For WP  and press Activate
*  After install, activate the plugin.

== Frequently Asked Questions ==

= 1. How can I add Logos to the Logo Showcase? =
For add logos on back-end, you can use from  Logos -> Add logo of Wordpress  left menu

= 2. What data can i add on each Logo ? =
Title, logos image, short description, link URL, link target, full description, website, email, tell, logo image gallery, social links (facebook, twitter, instagram, google plus, linked in) each logo

= 3. Is the logo showcase responsive? =
Yes, the logo showcase is responsive and work fine on every devices.

= 4. Is the logo showcase have documentation? =
Yes, you can view it from [here](http://ithemelandco.com/Plugins/Documentations/LogoShowcase/)     .
